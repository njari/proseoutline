
Visual Elements : 

File Explorer on the left displaying the current outlines. 
Clicking on an outline loads the main branch only. Other branches are not accessible from here. 

Editing space for text in the centre - copy/paste should work here seamlessly. 
Save Button should be on the top right corner. 


Flow 1 : 
 go through user's daily notes on obsidian and extract meaningful themes to suggest possible topics to write about. 


Flow 2 : 
 given a topic, either generated through flow 1 or user specified - generate an outline for an article with citations from the users vault. 


 Flow 3 : 
 Allow the user to view and edit the outline generated. Also - when the user saves - ask if it should overwrite current version or stored separately. If current version - simply make a commit on the current branch of the article else create a new branch.
